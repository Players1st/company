Players1stのミッション

Players1st（プレイヤーズファースト）は、アスリート、料理人、職人、アーティスト、社会的課題解決のために行動する人など、人々の心を揺さぶる人たちの価値を高め、より良く生きていくための環境を創ります。 将来、子どもたちが創造的な職業を安心して選べるように、世の中をより良く変えていきます。

事業内容

Players1st

スポーツファンディングサイト

Players1st（プレイヤーズファースト）は、アスリート、専門家、社会的課題解決のために行動する人が活動資金を募集するサービスです。月額スポンサー、メッセージ付き投げ銭、スポンサー限定投稿、スポンサーとのメッセージなどファンとつながり支援を募ります。

サービスサイト

コインパス

メッセージ付き投げ銭アプリ

コインパスはあらゆるお店にメッセージ付き支援金を直接届ける投げ銭アプリです。金額は500円から。メッセージを添えて素早く届けることができます。「思いをカタチに」そんな大切にしたい思いをカタチしたアプリです。

アプリ紹介サイト

フットサルメット

フットサル専用メンバー募集アプリ

フットサルで人を集めたい人と参加したい人をつなぐアプリです。メンバー募集は、日時、コートの場所、募集人数を指定して作成するだけ。参加したい人は、募集一覧から参加申し込みをするだけ。募集人数が集まったら開催決定です。広告なしのシンプルなデザインで簡単に使えます。

業務スタイル

一般的な業務形態は採用していません。「スポーツの力で世の中をより良くする」コンセプトに共感し、実行できるかどうかを大事にしています。

リモートワーク オフィスを持たないフルリモートワーク。現状メンバーは、東京、青森。基本オンラインコミュニケーションです。

雇用形態はもっと自由に 正社員、契約(個人事業主や法人との)、パート・アルバイトなど形態は自由。

拘束時間なし ミッションとタスクによってドリブンさせるスタイルで、業務時間や業務規程はなし。
